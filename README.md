### Flood Intelligence System

ระบบจัดการข้อมูลน้ำและน้ำท่วมจากแหล่งข้อมูลภายนอก เพื่อนำไปใช้กับ dashboard และ model prediction

```text
Flood Intelligence

├── Water Level Data
├── Dam / Reservoir Data
├── Province / Basin Boundary
├── Station Master Data
├── Latest Database
├── Historical Database
├── Data Quality Log
├── Prediction Input
├── AI Forecast Output
└── OpenLayers Map Dashboard
```

#### Key Features

* ดึงข้อมูล latest และ history
* จัดเก็บข้อมูลเป็น Excel database

* ใช้ข้อมูลเป็น input สำหรับ prediction model
* แสดงผลบนแผนที่ OpenLayers
* รองรับ risk level เช่น Normal, Watch, Warning, Critical
* ออกแบบให้ต่อกับ backend API และ dashboard ได้
